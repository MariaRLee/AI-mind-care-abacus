# 🧮 AI-Mind Care 算盤算力
## AI-Mind Care Abacus Intelligence

**一珠一念，撥出專注與算力。**  
**One bead at a time—build focus, calculation, and confidence.**

AI-Mind Care 算盤算力是一個可在手機、平板與電腦上使用的互動式模擬算盤，採用傳統「一上珠、四下珠」設計，讓使用者透過點擊或拖曳算珠，練習加法、減法、整數與小數計算。

AI-Mind Care Abacus Intelligence is an interactive virtual abacus based on the traditional one-upper-bead and four-lower-bead design. Users can tap or drag the beads to practise addition, subtraction, integer calculations, and decimal calculations.

## ✨ 主要功能｜Main Features

- 傳統模擬算盤：8 位整數＋2 位小數  
  Traditional virtual abacus with 8 integer places and 2 decimal places
- 支援加法與減法  
  Addition and subtraction exercises
- 三種難度：簡單、中等、進階  
  Three difficulty levels: Easy, Medium, and Advanced
- 可選整數、1 位或 2 位小數  
  Integer, 1-decimal-place, or 2-decimal-place modes
- 可選不同題數進行挑戰  
  Multiple question-count options
- 顯示題目進度、完成時間與答對題數  
  Displays question progress, completion time, and correct answers
- 完成挑戰後顯示正確率與成果  
  Shows accuracy and results after completion
- 依成果提供煙火、彩紙、星光或鼓勵動畫  
  Fireworks, confetti, stars, or encouraging visual effects based on results
- 支援手機、平板與桌面瀏覽器  
  Responsive design for mobile, tablet, and desktop browsers
- 不需安裝 App  
  No app installation required

## 🚀 使用方式｜How to Use

1. 選擇難度、小數位與題數。  
   Select the difficulty, decimal places, and number of questions.
2. 按下「開始挑戰」。  
   Press **Start Challenge**.
3. 第一個數字會預先顯示在算盤上。  
   The first number is preloaded on the abacus.
4. 點擊或拖曳算珠，完成計算。  
   Tap or drag the beads to complete the calculation.
5. 按下「提交答案」。  
   Press **Submit Answer**.
6. 完成所有題目後，系統會停止計時並顯示正確率、完成時間與成果動畫。  
   After all questions are completed, the timer stops and the system displays accuracy, completion time, and a result animation.
