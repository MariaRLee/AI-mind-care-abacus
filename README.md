# AI-Mind Care 算盤算力｜AI-Mind Care Abacus Intelligence

**版本｜Version: V24 One-URL Chinese/English Mobile-First Senior Friendly Edition**

這一版是針對先前手機版仍過小、算盤只佔半個畫面、雙語同屏過於擁擠等問題，重新進行的結構性調整。

## V24 主要更新

- **維持一個 URL**：仍使用同一個 `index.html` 與同一個 GitHub Pages 網址。
- **中英切換**：畫面一次只顯示一種語言，避免雙語同屏造成文字過小。
- **手機優先重新設計**：手機版將次要文字移出主畫面，讓算盤成為主體。
- **更大的手機版算盤**：手機上算盤使用更多垂直空間，提升銀髮使用者可讀性。
- **設定可收合**：手機版的難度、小數位、題數放入可展開的設定區。
- **簡化手機頁尾**：主畫面只保留短版 IP 提示與 Full Terms 連結。
- **保留原功能**：加減法、整數與小數、5 題 / 10 題、計時、正確率、完成動畫等功能仍保留。
