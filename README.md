# 🧮 AI-Mind Care 算盤算力
## AI-Mind Care Abacus Intelligence

**目前版本｜Current Version: V23.2 Mobile Fit + IP Protection Edition**

**一珠一念，撥出專注與算力。**  
**One bead at a time—build focus, calculation, and confidence.**

AI-Mind Care 算盤算力是一個可在手機、平板與電腦上使用的雙語互動式模擬算盤。使用者可點擊或拖曳算珠，練習加法、減法、整數與小數計算。
